# About

Discord Rich Presence for what you are reading on arXiv, powered by a browser extension and a small local helper.

Not affiliated with arXiv or Cornell; uses only public arXiv pages. Disclaimer: personal/educational use only; no affiliation or endorsement; bring your own icons/assets and avoid third-party trademarks.

## Install & Launch (GUI)
1) Install and start Paper Attention.

2) Load the browser extension
    - Chrome/Edge: Extensions > enable Developer Mode.
    - Click "Load unpacked" and point to the `extension/` folder.
    - Open any arXiv paper page; the GUI log should show incoming updates.

## Usage
- Keep Discord and the helper running.
- Visit an arXiv abstract or PDF page; Rich Presence updates automatically.
- Buttons are on by default; turn them off in the extension options if you prefer.

## Troubleshooting
- No update? Check the helper console for POST logs and ensure Discord is open.
